# Project Name: NFT Verification Bot for Discord

## Description
This project includes a Discord bot designed to verify NFT ownership using Ethereum blockchain technology. It integrates with Discord through commands and provides a secure way for users to connect their EVM compatible wallets and verify ownership of NFTs.

## Features
- Discord bot integration.
- Secure wallet connection.
- NFT verification using smart contracts.
- Interactive session management for user verification processes.

## Prerequisites
Before you begin, ensure you have met the following requirements:
- Node.js (v14.x or newer)
- npm (v6.x or newer)
- Discord account and a bot created on the Discord Developer Portal
